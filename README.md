# eMensageria

eMensageria - Sistema de Mensageria Opensource do eSocial <www.emensageria.com.br>
(em desenvolvimento)

## Versões compatíveis do eSocial

- S 1.0

## Requisitos de Sistema

- Python (versão 3.8.5);
- Django (versão 3.2);
- Postgres 12;
- outros requisitos no arquivo requirements.txt
